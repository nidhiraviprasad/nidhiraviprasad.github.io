My Journey

I first started programming when I joined my school's Girls Who Code club. I wrote my first "Hello, World" program in HTML and ended up creating my first website.

The more I learned about programming, the more interested I became. I started learning JavaScript on free online websites, and went on to learn Java and Python as well.

Currently, I am creating web apps, using machine learning, and learning about fullstack app development! I've created multiple projects with ML in Python using the libraries tensorflow, keras, tflearn, scikit, and others. I have also worked with multiple kinds of databases (Firebase & MongoDB) and frameworks (Flask/Django, ReactJS)!

My journey with programming is far from over and I am very excited to share what I learn!
